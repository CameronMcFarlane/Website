Optional Modpacks

While this server can be played on by a vanilla client, it is configured to support proximity voice chat - which is always more fun when everyone gets involved! To simplify installation, I've bundled the requirements into a modpack which is available for download below. If you're only interested in the proximity voice chat experience, then this is all you'll need.

mcfarlane-modpack-1.21.4

Modpack Installation

The following steps are for installing the mcfarlane-modpack.

  1. Close Minecraft and the Minecraft Launcher if either are running.
  2. Unzip the mcfarlane-modpack-1.21.4.zip file into a directory of your choosing.
  3. Run the fabric-installer-1.0.0.exe application. Ensure that 1.21.4 is selected as the Minecraft Version and the Create profile option is ticked. Hit the Install button. You should see a message shortly appear saying that the install was successful.
  4. Navigate to the .minecraft directory by using the Win + R shortcut, entering %appdata% into the Run window and hitting Ok.
  5. Inside the .minecraft directory, you should find a mods directory. Delete it, then copy the mods directory from the unzipped mcfarlane-modpack into the .minecraft directory.
  6. Open the Minecraft Launcher and select Minecraft: Java Edition. To the left of the Play button, select the fabric-loader from the dropdown list.
  7. Hit the Play button and accept the warning. Minecraft should launch successfully!

If you would also like to install the mcfarlane-modpack-plus, then continue with the steps below - provided that the mcfarlane-modpack has been installed successfully.

  1. Close Minecraft and the Minecraft Launcher if either are running.
  2. Unzip the mcfarlane-modpack-plus-1.21.4.zip file into a directory of your choosing.
  3. Navigate to the .minecraft directory - using the same method from Step 3 above if necessary.
  4. Copy the contents of the mods folder from the unzipped mcfarlane-modpack-plus into the mods folder inside .minecraft.
  5. Copy the shaderpacks folder from the unzipped mcfarlane-modpack-plus into the .minecraft directory.
  6. Open the Minecraft Launcher and select Minecraft: Java Edition. To the left of the Play button, select the fabric-loader from the dropdown list. Hit the Play button and wait for Minecraft to launch.
  7. From the main menu, select Options > Video Settings > Shader Packs and select the installed shader pack from the list.
  8. Join the server and play the game!
