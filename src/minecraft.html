Modpack

While this server can be played on by a vanilla (unmodded) client, I highly recommend you add the proximity voice chat mod to your client and join in on the fun. To help get you all set-up quickly, I've bundled a selection of mods into a pack which is available for download below.

⤵ faffpack-1.21.11

Entity Culling: More performance improvements by skipping rendering of entities you can't see.

...and a bunch of other mod dependencies you probably don't care about :)

Modpack Installation

Note: this will create a separate instance of Minecraft using Installations in the Minecraft Launcher. Any existing mods or worlds you have in your default installation will be left untouched.

The following steps assume that you're running Windows and have the Minecraft Launcher installed. If one of these is not the case, see the Help! section at the bottom of this page.

  1. Close Minecraft and the Minecraft Launcher if either are running.
  2. Navigate to the .minecraft directory by using the Win + R keyboard shortcut, entering %appdata% into the Run window and hitting Ok.
  3. Open the faffpack-1-1.21.11.zip by double clicking on it.
  4. Copy the .faff-1.21.11 and versions folders from the faffpack-1-1.21.11.zip into the .minecraft directory.
  5. Open the Minecraft Launcher and select Minecraft: Java Edition from the left sidebar.
  6. Click on the Installations tab at the top, followed by the New installation button. Pick an icon and enter a name of your choosing.
  7. Click the version dropdown, type "fabric" and select fabric-loader-0.18.2-1.21.11 from the list.
  8. Configure the game directory by clicking the browse button and selecting the .faff-1.21.11 folder inside your .minecraft directory and clicking the Upload button.
  9. In the bottom-right corner, untick the Launch after Install checkbox and click the Install button.
  10. If not already selected, find your new installation in the dropdown to the left of the Play button. Hit the Play button and accept the warning. Minecraft should launch successfully!

Modpack Uninstallation

  1. Close Minecraft and the Minecraft Launcher if either are running.
  2. Navigate to the .minecraft directory by using the Win + R keyboard shortcut, entering %appdata% into the Run window and hitting Ok.
  3. Open the .faff-1.21.11 folder and copy any worlds out of the saves folder which you wish to keep.
  4. Go back to the .minecraft directory and delete the .faff-1.21.11 directory.
  5. Open the Minecraft Launcher and select Minecraft: Java Edition from the left sidebar.
  6. Click on the Installations tab at the top. Hover over the installation you made for this server, select the three-dots icon on the right, and click Delete.
  7. Confirm the deletion in the dialog box that appears.
